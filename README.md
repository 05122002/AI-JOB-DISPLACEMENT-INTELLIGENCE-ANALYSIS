# AI-JOB-DISPLACEMENT-INTELLIGENCE-ANALYSIS
This project investigates the impact of Artificial Intelligence and automation on the global workforce across 15,000 job records spanning 8 industries, 9 countries, and 10 job roles from 2020 to 2026. Using Microsoft Power BI, a 4-page interactive dashboard was built covering automation risk, salary impact, reskilling urgency, skill gap.
